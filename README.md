# PPPK – Projektni zadatak

Dva projekta za kolegij "Pristup podacima iz programskog koda".

## project1-medical
Medicinski sustav (konzolna aplikacija). C# (.NET 8), Entity Framework Core (code-first), PostgreSQL.
Pokriva ishode I1, I4, I5.

## project2-bird-pipeline
Pipeline za obradu podataka o pticama. Python, MongoDB, MinIO.
Pokriva ishode I2, I3.

Baze se pokreću u Dockeru. Upute za pokretanje su u mapi svakog projekta.
